## 원리
  ### 1. BMP 스테가노그래피  
    BMP 이미지의 하나의 픽셀은 24비트의 RGB(각각 8비트) 정보로 이루어져있음.  
    각각의 RGB 정보의 LSB를 변형함으로서 메시지 삽입
  ### 2. JPG 스테가노그래피  
    JPG 이미지의 마지막에 EOI(FF D9)가 존재함  
    EOI 뒤에 있는 데이터는 무시되기 때문에 EOI 뒤에 메시지 삽입
## 사용법
    steg [-h filename "message" outputfile] [-r filename]
  * -h 메시지 숨기기 옵션
  * -r 숨겨진 메시지 출력 옵션
